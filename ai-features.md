# AI Features

Hackvertor integrates with Burp Suite's AI capabilities to generate code and learn encodings.

## Requirements

- Burp Suite Professional with AI features enabled
- AI credits (consumed when using features)

AI features are only available when Burp's AI API is accessible.

## Enabling AI Features

Go to **Hackvertor → Settings → AI**:

| Setting | Description |
|---------|-------------|
| Use AI to generate code | Enable code generation |
| Use AI to summarise custom tag code | Auto-describe tags |
| Use AI in tags | Allow AI-powered tags |
| Debug AI requests | Log AI conversations |
| Use AI to learn from Repeater | Pattern detection |
