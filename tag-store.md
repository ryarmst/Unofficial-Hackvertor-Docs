# Tag Store

The Tag Store provides community-contributed tags that can be installed with one click.

## Accessing the Tag Store

1. **Hackvertor menu → View tag store**
2. Or click "Open tag Store" button in the Tag store tab

## Installing Tags

1. Browse available tags in the list
2. Click on a tag to view its description and code
3. Click "Install" to add it to your custom tags

Installed tags appear in the **Custom** category.

## Updating Tags

If you install a tag and the store version is updated:
1. Delete your local version (List custom tags → Delete)
2. Reinstall from the tag store
